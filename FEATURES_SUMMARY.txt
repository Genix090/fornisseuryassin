┌─────────────────────────────────────────────────────────────────────┐
│  📊 MENU: FONCTIONNALITÉS AVANCÉES                                  │
└─────────────────────────────────────────────────────────────────────┘

  1. 📁 Exporter en CSV
     └─→ Exportez tous vos fournisseurs en format CSV
         Compatible Excel, Google Sheets, LibreOffice

  2. 📥 Importer CSV
     └─→ Importez des fournisseurs depuis un fichier CSV
         Vérification automatique des doublons

  3. ⭐ Noter Fournisseur
     └─→ Système de notation 5 étoiles
         • Qualité des produits
         • Livraison
         • Prix
         • Communication
         + Zone de commentaires

  4. 🔍 Filtre Avancé
     └─→ Filtrage multi-critères
         • Par nom
         • Par email
         • Par type de produits
         • Par adresse
         • Par note (min-max)

  5. 📝 Historique d'Activités
     └─→ Journal complet de toutes les actions
         Horodatage, utilisateur, description

  6. 📈 Statistiques Avancées
     └─→ Analyse complète des données
         • Total fournisseurs (actifs/inactifs)
         • Note moyenne
         • Meilleur fournisseur
         • Distribution par type
         • Distribution des notes

  7. 💾 Créer Sauvegarde
     └─→ Backup manuel avec timestamp
         Stockage dans dossier backups/

  8. ♻️ Restaurer Sauvegarde
     └─→ Restauration à partir d'une sauvegarde
         Liste de toutes les sauvegardes disponibles

┌─────────────────────────────────────────────────────────────────────┐
│  🎨 BOUTONS PRINCIPAUX                                              │
└─────────────────────────────────────────────────────────────────────┘

  • [Ajouter]      → Ajouter un nouveau fournisseur
  • [Modifier]     → Modifier le fournisseur sélectionné
  • [Supprimer]    → Supprimer un fournisseur (avec confirmation)
  • [Rechercher]   → Recherche rapide par ID/nom/email
  • [Trier]        → Tri par ID/Nom/Type/Rating
  • [Stat]         → Statistiques de base
  • [Export Pdf]   → Export PDF formaté professionnel
  • [Ref]          → Actualiser la vue
